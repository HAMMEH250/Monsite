# Plateforme EEUCI - DISTRICT DE KOTOKODJI

## Description
Plateforme de gestion complète pour EEUCI - DISTRICT DE KOTOKODJI avec système d'authentification, tableau de bord, gestion des documents, activités et membres.

## Fonctionnalités
- Système d'authentification (connexion/inscription)
- Tableau de bord avec statistiques
- Gestion des actualités et notifications
- Gestion des documents et cours
- Suivi des activités
- Gestion des membres

## Technologies utilisées
- HTML5
- CSS3 (Bootstrap 5.3.0)
- JavaScript
- Chart.js pour les graphiques
- Font Awesome pour les icônes

## Installation
1. Téléchargez le fichier HTML
2. Ouvrez-le dans votre navigateur web
3. Créez un compte ou utilisez les identifiants de test

- Responsable: username=responsable, password=resp123
